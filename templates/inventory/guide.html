How the module works
The inventory module tracks reagents and consumables used in your lab, radiology, and other departments. Stock goes up when you receive a delivery and comes down automatically when you upload a LabSmartLIS export — it matches each test performed to the reagents it consumes.

{{ step(1, 'First-time Setup', 'MDS / Accountant', '') }}

The system ships with 132 reagents and consumables and 21 packages pre-loaded from your reagent list. You do not need to enter them manually. On first login to the Inventory section, check that they appear under Items Catalogue.

Checklist before you start:
☐  Go to Items Catalogue — confirm items loaded
☐  Go to Packages — confirm packages loaded
☐  Set reorder levels on high-turnover items (edit each item)
☐  Do an initial stock receive to enter your current physical stock quantities
☐  Map each test to the reagents it uses (Step 3)
{{ step(2, 'Item Catalogue', 'MDS / Accountant', '') }}

The catalogue lists every item you want to track. Each item has a category (Lab Reagent, Lab Consumable, X-ray Consumable, etc.), a unit (strip, vial, ml, piece…), optional pack size, and a reorder level that triggers low-stock alerts.

➕ Add a single item
1. Go to Items Catalogue
2. Click Add Item
3. Fill in name, category, unit, price, reorder level
4. Save — item appears in the list immediately
📤 Bulk add via CSV
1. Click Export CSV to get current list
2. Edit or add rows in Excel/Sheets
3. Save as CSV
4. Click Import CSV and upload
Existing items (matched by code or name) are updated; new rows are added.
{{ tip('Set a reorder level on every high-turnover reagent. The dashboard will highlight anything running low so you can reorder before you run out.') }}
{{ step(3, 'Test Catalogue & Reagent Mapping', 'MDS / Accountant / Lab Staff', '') }}

The test catalogue connects each investigation (e.g. Full Blood Count) to the reagents it consumes. This is what makes automatic depletion possible — when LabSmartLIS says 50 patients had FBC, the system looks up FBC's reagent mapping and subtracts the right quantities.

How to map a test to its reagents:
1. Go to Test Catalogue → click Add Test
2. Enter the test name exactly as it appears in LabSmartLIS (the LIS Name field)
3. Save — then click the test name to open its detail page
4. Under Reagent Mappings, click + Add Reagent
5. Select the item and enter qty per test (e.g. 1 strip, 0.5 ml)
6. Repeat for all reagents this test uses
{{ note('The LIS Name must match what LabSmartLIS exports. If depletion is not working for a test, check spelling here first — it uses fuzzy matching but an exact match is most reliable.') }} {{ tip('Start with your top 10 highest-volume tests. You do not need to map every test before you begin — unmapped tests are flagged for review after each LIS upload.') }}
{{ step(4, 'Packages & Panels', 'MDS / Accountant', '') }}

Packages are multi-test panels that appear as a single line in LabSmartLIS (e.g. SURE COMPLETE HEALTH PACKAGE). When the LIS upload encounters a package name, it automatically expands it into its constituent tests and depletes reagents for each one.

How to configure a package:
1. Go to Packages — the 21 standard packages are already created
2. Click a package name to open it
3. Under Constituent Tests, tick all the tests included in this panel
4. Save — the package will now expand correctly on LIS upload
{{ tip('Check the LIS Name field on each package matches exactly what appears in your LabSmartLIS export. Edit it if needed.') }}
{{ step(5, 'Receiving Stock (Deliveries)', 'Accountant / Lab Staff', '') }}

Every time reagents or consumables arrive from a supplier, record the receipt here. This adds the quantity to the stock level for the chosen branch (or Central if no branch selected).

📝 Single item receipt
1. Go to Receive Stock
2. Select the item, branch, and enter quantity
3. Optionally add unit cost, batch number, expiry date
4. Click Record Receipt
📤 Bulk receive via CSV
1. Click Download Template for a pre-filled CSV
2. Fill in quantities, costs, and batch numbers
3. Click Bulk Upload CSV and upload the file
Useful after a large delivery with many different items.
{{ note('Stock levels are per-branch. If your lab at Ijofi receives reagents, select "Ijofi" as the branch. Leave Branch blank to record to Central Stock.') }}

{{ step(7, 'Low-Stock Alerts', 'All roles', '') }}

Any item whose current stock falls below its Reorder Level appears as a red alert on the Inventory Dashboard. Check this daily to catch shortages before they disrupt operations.

To set or change a reorder level:
1. Go to Items Catalogue
2. Click the item name → click Edit
3. Enter a value in Reorder Level (in the same unit as the item)
4. Save — the item will now appear on the dashboard when stock drops below this level
{{ tip('A reorder level of 10 on an item measured in strips means "alert me when fewer than 10 strips remain."') }}
{{ step(8, 'Reports & Adjustments', 'MDS / Accountant', '') }}
📊 Consumption Report
Go to Reports → select a date range and branch → view total received vs consumed per item, with estimated cost. Use this monthly to track reagent spend.
⚙️ Stock Adjustments
After a physical stock count, correct the system using Adjust Stock. Choose Adjustment to correct a variance or Write-off to record damaged/expired items. Every adjustment is logged with a timestamp and user.
{{ tip('Do a physical count every Monday morning (high-turnover items) and a full count at end of month. Use Stock Adjustment to reconcile any differences before the next LIS upload.') }}
