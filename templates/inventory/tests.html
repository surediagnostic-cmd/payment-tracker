{# Import Aliases Modal #}
Import LIS Name Aliases
Attach the HMO/branch billing variants (e.g. MP IKEJA, MP NEM) to existing tests, tag each with its branch, and optionally set that branch's price.

Columns (first row = header):
Test · Alias · Branch · Set Price · Fee

• Test — matched to an existing test by LIS name, then display name, then closest fuzzy match.
• Alias — one LabSmart name per row (legacy pipe-separated Aliases column still accepted).
• Branch — the branch this variant belongs to; a branch-labelled name → that branch, everything else → Ilasa.
• Set Price = yes writes Fee as that branch's price — but only when the test matches exactly (loose matches are reported, never mis-priced).
• Names already mapped elsewhere are left untouched and reported.
